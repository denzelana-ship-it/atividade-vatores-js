
# Atividade de Vetores em JavaScript

## Descrição

Este repositório contém a resolução de três exercícios utilizando vetores (arrays) em JavaScript. Os programas foram desenvolvidos no Visual Studio Code e utilizam estruturas de repetição para leitura, armazenamento e processamento de dados.

## Tecnologias Utilizadas

* JavaScript
* Visual Studio Code
* Git
* GitHub
* Node.js
* Prompt-sync

## Exercício 1 - Maior Idade

O programa lê a idade de 4 pessoas e armazena os valores em um vetor.

### Funcionalidades

* Armazena as idades em um vetor.
* Identifica a maior idade digitada.
* Exibe a posição em que a maior idade foi informada.

## Exercício 2 - Funcionárias com Salário Acima de R$ 5.000

O programa lê o nome, sexo e salário de 5 funcionários e armazena as informações em três vetores.

### Funcionalidades

* Armazena nomes, sexos e salários em vetores.
* Percorre os vetores utilizando laço de repetição.
* Exibe apenas as funcionárias do sexo feminino que possuem salário superior a R$ 5.000,00.

## Exercício 3 - Controle de Alturas

O programa permite cadastrar nomes e alturas de várias pessoas.

### Funcionalidades

* Armazena nomes e alturas em vetores.
* Permite inserir quantos registros desejar.
* Encerra o cadastro quando o usuário escolher parar.
* Exibe a maior altura e a pessoa correspondente.
* Exibe a menor altura e a pessoa correspondente.

## Como Executar

1. Instale o Node.js.
2. Instale a biblioteca prompt-sync:

```bash
npm install prompt-sync
```

3. Execute o arquivo desejado:

```bash
node exercicio1.js
```

```bash
node exercicio2.js
```

```bash
node exercicio3.js
```
